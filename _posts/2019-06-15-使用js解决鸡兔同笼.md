  那天上完数学课,突发奇想:做个鸡兔同笼计算器.
  先用假设法:
  ​```
  头x2=x
  (脚-x)÷2=兔子只数
  头-兔子只数=鸡只数
  ​```
  试了一下,不行.
  暴力破解:
  ```html
<!DOCTYPE html>  
<html>  
    <head>  
        <meta charset="UTF-8">  
        <title>鸡兔同笼</title>  
    </head>  
    <center id="b">  
        <input type="number" id="head" placeholder="头"/>  
        <br/>  
        <input type="number" id="foot" placeholder="脚"/>  
        <br />  
        <input type="button" onclick="ccccc()" value="计算"/>  
    </center>  
    <center id="a">  
          
    </center>  
        <script>  
            function ccccc() {  
                var a=document.getElementById("a");  
                var b=document.getElementById("b");  
                var head = document.getElementById("head").value;  
                var foot = document.getElementById("foot").value;  
                var chicken_head = 0;  
                var rabbit_head = 0;  
                for ( chicken_head = 0; chicken_head <= head; chicken_head++) {  
                    rabbit_head = head - chicken_head;  
                    if (2 * chicken_head + 4 * rabbit_head == foot) {  
                        a.innerHTML="鸡有" + chicken_head + "只" + "</br>"+"兔有" + rabbit_head + "只";  
                        b.style.display="none";  
                    }  
                }  
            }  
  
        </script>  
</html>  
```
